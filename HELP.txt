Installing and Running Snowman

System Requirements

This program is run in the terminal. If you're reading this, your computer very likely has a terminal.

In order to run this program, you need Python 3. If you do not have Python 3, go to https://installpython3.com to download.

Installation

Download the files to a directory on your hard drive.

Running the Program

Open your terminal, and navigate to the folder contaning this program.
Enter the following text, followed by Enter/Return:
chmod +x snowman.sh

The enter the following text, followed by Enter/Return:
./snowman.sh

To open the ABOUT file, enter the following text, followed by Enter/Return:
./snowman.sh about
